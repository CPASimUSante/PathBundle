# PathBundle


## Installation

Install with composer :

    $ composer require "innova/path-bundle" "1.*" 
    $ php app/console claroline:plugin:install InnovaPathBundle

## Uninstall 

    $ php app/console claroline:plugin:uninstall InnovaPathBundle
